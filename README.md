# PS Hotkey Enforcer

**在中文輸入法下，讓 Photoshop 快捷鍵正常運作。**

**[程式介紹頁面](https://pshotkeyenforcer.blogspot.com/)**

---

## 問題

在 Windows 上使用中文輸入法（注音、倉頡、大易等）時，Photoshop 的單鍵快捷鍵會被輸入法攔截——按 `B` 想切筆刷，結果跳出注音選字框；按 `V` 想切移動工具，卻開始打字。每次都要手動切換成英文輸入法，用完再切回來，畫圖的節奏被反覆打斷。

Mac 不會有這個問題，但 Windows 就是會。

## 解決方式

PS Hotkey Enforcer 在背景監聽鍵盤事件，當偵測到目標應用程式為前景視窗時，將按鍵繞過輸入法直接送給應用程式。你不需要切換輸入法，快捷鍵就是能用。

基於 AutoHotkey v2 開發，透過 `ControlSend` 繞過 IME 攔截，原理單純、行為透明。

## 功能

- **圖形化設定介面** — 所有設定用滑鼠完成，不需要改程式碼
- **一鍵暫停 / 啟用** — 預設 `Ctrl+F2` 切換，需要打字時暫停，畫圖時開啟
- **不只 Photoshop** — 可自行加入任何受輸入法干擾的應用程式（繪圖軟體、3D 建模、DAW 等）
- **自訂修飾鍵覆蓋** — Shift、Alt、Ctrl 及任意組合，依個人習慣調整
- **按鍵偵測設定熱鍵** — 點一下偵測按鈕，直接按鍵盤設定，不用手打 `Ctrl+F2`
- **免安裝** — 解壓縮直接執行，不寫入登錄檔

### 關於設定檔

程式會在同資料夾自動產生 `PS_Hotkey_Enforcer.ini` 儲存你的設定。搬移程式時記得一起帶走，設定才會保留。沒帶走也沒關係，重新設定一下就好。想還原預設值的話，直接刪除 `.ini` 檔再重開程式即可。

## 首次執行

### Windows SmartScreen 警告

本程式是個人開發，沒有購買數位簽章，首次執行時會跳出藍色「已保護您的電腦」提示。這不代表程式有問題，只是 Windows 不認識發行者。

排除方式：點擊「其他資訊」→「仍要執行」。

### 防毒軟體誤判

這個工具在系統底層攔截鍵盤訊號再轉發給目標視窗，這種行為跟鍵盤監控程式的原理相似，因此部分防毒軟體（例如 Windows Defender 的 Wacatac 標籤）可能會誤判。這是 AutoHotkey 類工具的常見現象。

程式已上傳至 [VirusTotal](https://www.virustotal.com/) 掃描，絕大多數引擎判定為安全。
